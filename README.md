## 👋 Visão Geral

Portfólio pessoal de **Cauã**, 18 anos, estudante de **Ciência da Computação na UVV — ES**.

Construído do zero para reunir minha trajetória, stack técnico e projetos em um só lugar. Sem frameworks JavaScript — só HTML, CSS e Bootstrap fazendo o trabalho pesado. O destaque fica por conta do **tema claro/escuro implementado puramente em CSS**, usando um checkbox oculto como gatilho, sem uma linha de JS.

## 🚀 Instalação

**Pré-requisitos:** um navegador moderno. Só isso.

**Opção 1 — Abrir direto:**
```bash
git clone https://github.com/ccauxdev/<repositorio>
# Abra o index.html no navegador
```

**Opção 2 — Servidor local (recomendado):**
```bash
# Python
python -m http.server 8000

# Node.js
npx http-server
```
Acesse em `http://localhost:8000`
